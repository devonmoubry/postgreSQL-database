# PostgreSQL Database - Todo List

BUILD STEPS:
- `mkdir <project-name>`
- `cd <project-name>`
- `touch README.md`
- `touch todo.sql`
- `createdb todolist`
- `psql todolist`
- `\d` (check yourself)
- `\q` (make it stop)
- `psql -f todo.sql todolist` (run file in terminal)
- google all the things

PROJECT FEATURES:
- `CREATE TABLE`
 - `id`
 - `title`
 - `details`
 - `priority`
 - `created_at`
 - `completed_at`
- `INSERT INTO`
- `SELECT`
- `UPDATE`
- `DELETE FROM`
